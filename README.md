# 🛒 Store E-Commerce Sales Dashboard | Power BI

An interactive Power BI dashboard analyzing e-commerce store performance across sales, profit, quantity, and customer segments.

## 📌 Objective
Provide business stakeholders a visual overview of sales trends, category performance, and regional distribution to support data-driven decisions.

## 🗂️ Files
| File | Description |
|------|-------------|
| `ecommerce_sales_data.pbix` | Main Power BI report file |
| `Orders.csv` | Primary dataset — order-level transaction data |
| `Details.csv` | Secondary dataset — product & category details |
| `dark-gradient.jpg` | Background image used in dashboard |

## 📊 Dashboard Features
- Total Sales, Profit, Quantity & AOV KPI cards
- Sales & profit trend over time
- Top performing product categories & sub-categories
- State-wise revenue breakdown (map visual)
- Customer-wise sales analysis
- Filters by Quarter, State, and Category

## 🛠️ Tools Used
- Power BI Desktop
- DAX (calculated measures & KPIs)
- Excel / CSV (data source)

## 💡 Key Insights
- Clothing is the highest quantity-sold category
- Maharashtra & Madhya Pradesh lead in regional sales
- Q4 shows peak sales performance
